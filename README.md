# Optimising-Bat-Friendly-Curtailment-Algorithms-for-Wind-Turbines-

Script 01 allows to simulate an algorithm-based curtailment on wind turbines for which operation and bat activity data are availalbe, using different model training methods. This script uses the dataset "data_2_5_t1.csv" available here: https://data.indores.fr/dataset.xhtml?persistentId=doi:10.48579/PRO/DJD158

Script 02 allows to compute from the output of the script 01 the algorithm efficiency (bat protection and energy loss) using different methods of defining curtailment threshold.

Script 03 allows to compute a plot algorithm efficiency over the year from the output of the script 01.
